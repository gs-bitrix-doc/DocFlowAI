## Проблемы

### КРИТИЧНО — нужно чинить

#### 1. `тип` не переведён в заголовках таблиц
Protector защищает весь инлайн-код, включая `` `тип` `` — а это не технический термин, а подпись колонки.

| Было | Должно быть |
|---|---|
| `` `тип` `` | `` `type` `` |

Встречается на каждой странице документации. **Нужно исключить из защиты или добавить словарную замену после перевода.**

#### 2. Заголовки битрикс-блоков не переводятся
Весь `{% note ... %}...{% endnote %}` уходит в плейсхолдер целиком — включая текст заголовка.

| Было | Должно быть |
|---|---|
| `{% note warning "Развитие метода остановлено" %}` | `{% note warning "Method development stopped" %}` |
| `{% note tip "Связанные методы и темы" %}` | `{% note tip "Related methods and topics" %}` |

**Нужно либо защищать только теги, но не текст заголовка, либо переводить заголовки отдельным шагом.**

#### 3. `ниже` не переведено внутри текста
Строка: `Structure described [ниже](#deal)` — слово "ниже" осталось русским.
Это скорее всего попало в плейсхолдер как часть ссылки.
Должно быть: `Structure described [below](#deal)`

---

### СРЕДНЕЕ — влияет на качество

#### 4. Метки include-файлов не переведены
```
{% include [Сноска о параметрах](../../../_includes/required.md) %}
{% include [Сноска о примерах](../../../_includes/examples.md) %}
{% include notitle [обработка ошибок](../../../_includes/error-info.md) %}
{% include [системные ошибки](./../../../_includes/system-errors.md) %}
```
Текст в `[квадратных скобках]` — это подпись, она отображается на странице. Сейчас эти блоки уходят в плейсхолдеры целиком.

#### 5. Строки-примеры в JSON не переведены
Внутри JSON-блока кода значения строк остались русскими:
```json
"TITLE": "Новая сделка #1"
"COMMENTS": "[B]Пример комментария[\/B]"
"ADDITIONAL_INFO": "Дополнительная информация"
"UF_CRM_...": "Привет мир!"
```
Это пример ответа API — спорный момент, переводить или нет. **Для английской доки логичнее перевести.**

---

### НИЗКОЕ — ложные срабатывания валидатора

Валидатор ругается на «непереведённые» термины: `сделка`, `источник`, `метод`, `параметр`, `сортировка`, `постраничная навигация`.
Фактически все они переведены — `deal`, `source`, `method`, `parameter`. Валидатор ищет подстроку без учёта контекста.
`сортировка` и `постраничная навигация` в этом файле просто не встречаются.
**Логику валидатора словаря нужно улучшить — проверять только если слово есть в тексте.**
